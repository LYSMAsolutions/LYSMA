# LYSMA Hub

Site vitrine statique intégré à la monorepo LYSMA.
Sans fautes de grammaire, fautes de syntaxes, fautes de conjugaison, toutes fautes. Bien écrire chaque mots et surtout que ça ai du sens

## Commandes

- `pnpm --filter site-vitrine-lysma-hub run dev` lance un serveur local.
- `pnpm --filter site-vitrine-lysma-hub run check` vérifie les erreurs HTML simples.

Le site reste volontairement statique : HTML, CSS et JavaScript sans build lourd.

## METIER

le site doit pouvoir avoir afficher les aperçu de chaque site vitrine qu'il y aura dans LYSMA/apps/site-vitre
